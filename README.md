![image](https://github.com/Tao-Dylan-CH/ZBlog/assets/81242825/518966ea-0a27-4e7d-a93d-a41dd860f306)# ZBlog
个人博客。
前端和后端。
根据类型和标签管理博客。
# 技术栈
- 数据库 ： Mysql
- MyBatis Plus
- MVC : Spring MVC
- SpringBoot
- 模板引擎: Thymeleaf
- UI : Semantic UI
